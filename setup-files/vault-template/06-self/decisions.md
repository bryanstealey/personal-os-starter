# Decisions Log

Consequential decisions only. This is NOT a daily-choices log — it is for decisions that shape direction, commitments with long tails, or reversible choices you want to revisit later.

## Format

```
## YYYY-MM-DD — [Decision name]

**Context:** [What was the situation that required a decision?]
**Options considered:** [What was on the table?]
**Chosen:** [What was decided, stated as a commitment.]
**Why:** [The reasoning — specifically the tradeoffs accepted.]
**Revisit:** [Date or condition that should trigger re-evaluation, or "stable" if not expected to reopen.]
**Source:** [Session handoff, conversation, or decision moment.]
```
